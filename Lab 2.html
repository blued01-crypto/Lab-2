2. What is HTML?
HTML stands for HyperText Markup Language. To create the structure and content of web pages.

3. How does a browser request a web page?
When you type a URL into your browser, it sends an HTTP request to the web server. The server then responds by sending the HTML file back, which the browser displays.

4. Difference between regular and empty tags?
Regular tags have an opening and closing part like <p> </p>. Empty tags do not have content and are self-closing, like <br>.

5. What is white space collapsing?
White space collapsing means that multiple spaces, tabs, or line breaks in HTML are treated as a single space when rendered by the browser.
